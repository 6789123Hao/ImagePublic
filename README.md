# ImagePublic

Repository for future web-based game/project prototype asset drops.

## Folder structure

```text
ImagePublic/
├── drops/
│   ├── web-games/      # Assets tied to web game projects
│   └── prototypes/     # Assets tied to prototype projects
├── shared/
│   ├── images/         # Reusable images/textures/backgrounds
│   ├── audio/          # Reusable SFX/music loops/stems
│   ├── fonts/          # Reusable typography assets
│   ├── ui/             # UI kits/icons/hud elements
│   └── 3d/             # Reusable 3D models/materials
├── incoming/           # Temporary staging area before sorting
├── archive/            # Retired/old assets
└── tools/
    └── templates/      # Optional naming/package templates
```

## Suggested naming convention for drop folders

Use timestamped folders under `drops/web-games/` or `drops/prototypes/`:

- `YYYY-MM-project-name`
- Example: `2026-05-space-runner`

This keeps uploads chronological and easy to search.
